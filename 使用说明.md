## 快速开始

### 首次使用

1. 打开软件 → 自动进入**首次向导**
2. 选择**性别** → 填写**身高/体重** → 系统自动计算每日推荐饮水量
3. 选择常用**水杯**（最多2个预设杯 + 可添加自定义水杯）
4. 点击「✨ 开始喝水」完成设置

> ⚠️ 首次使用会自动请求**系统通知权限**，建议允许以便接收喝水提醒

### 窗口操作

| 操作 | 方式 |
|------|------|
| 拖动窗口 | 按住顶部标题栏拖拽 |
| 隐藏到托盘 | 点击右上角 **✕** 按钮 |
| 恢复窗口 | 双击或左键点击托盘图标 |
| 退出程序 | 右键托盘 → 「🚪 退出程序」 |
| 切换主题 | 点击 **🌙 / ☀️** 按钮切换暗色/亮色 |

## 喝水打卡

### 快速打卡

首页预设了常用水杯按钮，点击即可记录：

```
[小茶杯 250] [矿泉水 500]   ← 点击自动记录
```

- 每次打卡有 **1.5 秒冷却**，防止误触
- 打卡后水杯有**弹跳动画**
- 达到目标时弹出庆祝效果 🎉

### 自定义水杯

点击快速打卡区的 **＋** 按钮：

1. 输入杯名（如"保温杯"）
2. 输入容量（ml）
3. 点击添加

支持无限个自定义水杯，可在设置页编辑/删除。

### 今日记录

点击「📝 今日记录」卡片 → 弹出今日所有喝水记录详情。

### 重置今日

点击底部 **🔄** 按钮 → 确认后清空今日所有记录。

## 统计数据

进入方式：点击标题栏 **📈**

### 今日圆环

- 显示今日完成百分比
- 未达标：蓝色，显示还差多少ml
- 已达标：绿色，显示 ✅

### 三栏卡片

| 数据 | 说明 |
|------|------|
| 🔥 连续达标 | 连续每天有喝水的天数 |
| 🏆 累计喝水 | 总共有喝水记录的天数 |
| 💧 今日已喝 | 今日总饮水量 |

### 本周柱状图

- **绿色柱子** = 当日达到目标 ✅
- **蓝色柱子** = 当日未达标，下方显示还差多少 ml
- 柱子高度按日饮水量比例显示
- 周末有单独标记 🏠

### 饮水分布

按水杯名称统计今日各杯饮水量，以条形图展示。

## 皮肤系统

进入方式：点击标题栏 **🎨**


| 操作 | 效果 |
|------|------|
| 点击动画 | 播放一次 |

| 每次喝水 | 自动播放一次动画 |
| 达到目标 | 彩纸庆祝动画 🎊 |

| 方式 | 消耗 | 说明 |

| 免费单抽 | 每天 1 次免费 | 每天刷新 |

## 设置

进入方式：点击标题栏 **⚙️**

### 个人资料

- 编辑性别、身高、体重
- 体重支持 kg / 斤 切换
- 修改后自动重新计算每日目标

### 每日目标

- 显示推荐目标值
- 可用 ±50/±100 按钮微调
- 范围：500ml ~ 5000ml

### 喝水提醒

| 设置项 | 说明 |
|--------|------|
| 🔔 提醒开关 | 开启/关闭所有提醒 |
| 间隔时间 | 15~180 分钟（默认45分钟） |
| 免打扰 | 设置时间段内不提醒 |
| 系统通知 | 进入电脑通知设置 + 测试按钮 |

**系统通知**：点击「测试」按钮检查系统通知是否正常工作。如未开启，点击「进入电脑通知设置」跳转到 Windows 设置页手动开启。

### 水杯管理

- 管理预设水杯（编辑/隐藏）
- 添加自定义水杯
- 支持增删改操作

### 数据管理

| 功能 | 说明 |
|------|------|
| 💾 导出数据 | 导出全部数据为 JSON（记录/设置/皮肤） |
| 📥 导入数据 | 导入之前导出的 JSON 文件恢复数据 |
| 🗑️ 清除所有数据 | 清空全部数据回到首次使用状态 |

## 常见问题

### Q: 为什么通知显示 "electron.app.吨吨吨"？
A: 便携版/免安装版运行时，Windows 会显示这个前缀。**安装安装包版本**（吨吨吨_安装包_1.0.0.exe）后即可显示为"吨吨吨"。

### Q: 如何备份数据？
A: 设置 → 数据管理 → 💾 导出数据，保存 .json 文件。重装后点 📥 导入即可恢复。

### Q: 皮肤动画怎么不动？
A: 首次装备皮肤需要加载动画数据（一秒左右）。如果动画不动，点击一下动画区域触发播放，或点击 🔁 开启循环模式。

### Q: 跨天数据会丢失吗？
A: 不会。所有数据保存在本地 localStorage，关闭软件不丢失。导出备份后可迁移到其他电脑。

### Q: 如何在浏览器中测试？
A: 直接双击 `index.html` 可在浏览器中运行（通知功能有限制，推荐用 Electron 版）。
